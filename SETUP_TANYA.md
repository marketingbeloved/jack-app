# 🐾 Jack — установка для Тани (Tobydic)

Привет! Это контент-хаб Джека. Ты будешь запускать его **локально у себя на маке** —
так Джек пишет ТЗ через наш корпоративный Claude (его «мозг»), а всё, что ты сохранишь,
видят остальные в общем облаке.

> Тебе НЕ нужно ничего покупать. Нужны только: эта папка, Python и вход в Claude.

---

## 1. Открой папку в VS Code
1. Распакуй архив `jack-app.zip` куда удобно (напр. в `Documents`).
2. В VS Code: **File → Open Folder…** → выбери папку `jack-app`.
3. Открой встроенный терминал: **Terminal → New Terminal**.

## 2. Установи Python-окружение
В терминале по очереди (скопируй целиком каждую строку):

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## 3. Войди в Claude (мозг Джека)
Джек генерит ТЗ через Claude CLI. Нужно войти **под нашим общим аккаунтом** (тот же,
что у Дарьи). Если Claude CLI ещё не стоял:

```bash
# установка (один раз):
npm install -g @anthropic-ai/claude-code
# вход — откроется браузер, залогинься под общим аккаунтом:
claude login
```

Проверь, что вход прошёл: команда `claude` должна открыть чат, выйти — `/exit`.

## 4. Запусти приложение
```bash
source .venv/bin/activate   # если терминал новый
streamlit run app.py
```
Откроется браузер на `http://localhost:8501`. Пароль на вход: **belovedjack2026**

---

## 5. Как «обучать» Джека под Tobydic
- В шапке слева переключи **Brand → Tobydic**.
- Джек уже знает Tobydic из файла `cache/notion_briefs_tobydic.json` (это примеры
  прошлых ТЗ — его «насмотренность»). Чем больше там хороших примеров, тем лучше ТЗ.
- Во вкладке **Content Plan** заполни даты/темы Tobydic (сейчас там пусто) —
  напиши Дарье или мне, добавим ячейки, как у BelovedPets.
- В каждой ячейке: поле «что хочешь» → Джек пишет ТЗ под твоё пожелание;
  блок «💬 Обсудить с Джеком» → он отвечает по-человечески и уточняет.

Всё, что сохраняешь, сразу уходит в общее облако (Supabase) + бэкап на твоём маке —
не потеряется.

---

## Если что-то не так
- **`command not found: streamlit`** → ты не активировала окружение: `source .venv/bin/activate`.
- **ТЗ не генерится / «Jack offline»** → не выполнен вход в Claude: `claude login`.
- **Не сохраняется у всех** → проверь, что файл `.streamlit/secrets.toml` на месте
  (он в архиве, не удаляй).
- Любая ошибка на экране — пришли скрин Дарье/Claude, разберём.

> ⚠️ Файл `.streamlit/secrets.toml` содержит ключи нашей общей базы и пароль —
> никуда за пределы команды его не выкладывай.
